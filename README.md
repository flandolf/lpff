# Lichess Puzzle FEN Finder (LPFF)
This script finds the FEN of a puzzle on lichess.org. It is useful for ~~cheating~~ **solving puzzles**.
## Dependencies
- Python 3
- chess
- colorama
- pyperclip
## Usage
- Run `python3 export_board.py` in the terminal.
